Data cleaning in SQL

In this project, I imported a CSV file into MySQL and created a reference table to avoid issues and reduce mistakes. I followed a structured data cleaning process: first, I removed duplicates using Common Table Expressions (CTEs). Next, I standardized the data by addressing inconsistencies such as spelling errors with UPDATE and SELECT statements. I then removed null or blank values using window functions. Finally, I simplified the dataset by dropping unnecessary columns with ALTER and DROP commands. This comprehensive approach ensured that the data was clean, consistent, and ready for analysis..
